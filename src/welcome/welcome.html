Sidetabs

How to use

Press Ctrl + Alt + R to toggle Sidetabs in the sidebar.

You can also click the Sidetabs icon in the toolbar to toggle Sidetabs.

You'll probably want to keep the sidebar on the left side of your browser window, but if you'd like to switch it to the right, click on the sidebar header, and click "Move Sidebar to Right".

What can Sidetabs do?

Sidetabs is intended as a replacement for regular Firefox tabs. It can therefore do almost everything that regular tabs can.

Tabs can be rearranged, closed, restored, and new tabs opened. Right-clicking a tab will display additional options.

Sidetabs will adapt to browser themes, to match the style of your regular tabs. Custom themes are supported.

Extra improvements

If you're looking to switch to the sidebar tabs lifestyle, there are a few additional tweaks you'll probably want to make.

In order to make a few cosmetic changes to Firefox, you'll need to edit your browser's userChrome.css file. You can use this file to add custom styles to your browser.

To edit your userChrome.css file, go to the Firefox Application Menu (The hamburger menu in the upper-right), and go to Help > More troubleshooting information. From there, scroll down to the Profile Folder section, and click to open it in your file explorer. Inside the profile folder, go to chrome/userChrome.css. Create this file if it doesn't exist already. This is the file that we will edit to add custom styles to Firefox.

  • To hide the default Firefox tabs, add this snippet:

    #TabsToolbar {
  display: none;
}

  • To hide the large "Sidetabs" header at the top of the sidebar, add this:

    #sidebar-header {
  display: none;
}
